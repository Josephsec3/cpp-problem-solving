# C++ Problem Solving & Algorithms

## 📝 Project Overview
This repository contains my solutions to various algorithmic problems and data structure challenges implemented in **C++**. It demonstrates my understanding of loops, vectors, and competitive programming techniques.

## 🚀 Key Features of the Code
* **Fast I/O Optimization:** Uses `ios_base::sync_with_stdio(false); cin.tie(nullptr);` to speed up input/output operations, which is essential for performance-critical tasks.
* **Array/Vector Manipulation:** Implements reverse traversal (`for (int i = n-1; i >= 0; i--)`) to optimize the state checking logic.
* **Efficient Complexity:** Processes queries dynamically within $O(N)$ time complexity per test case.
